## Descriere Proiect:

Tomotiv.ro este o platformă de e-learning, concepută pentru a oferi utilizatorilor posibilitatea de a se înscrie la diverse cursuri disponibile pe platformă. Această platformă permite administrarea eficientă a cursurilor și a utilizatorilor, oferind funcționalități extinse pentru gestionarea conținutului și a utilizatorilor.

Utilizatorii pot naviga și se pot înscrie la cursuri, având posibilitatea de a accesa materiale video, imagini sau texte. De asemenea, platforma permite administrarea cursurilor prin intermediul unui panou de control dedicat, unde administratorii pot gestiona conținutul, utilizatorii și alte aspecte ale platformei.

Un aspect important al platformei este posibilitatea de a acorda roluri diferite utilizatorilor, inclusiv rolul de instructor. Instrucții pot adăuga propriile cursuri pe platformă, împărtășind cunoștințele lor cu comunitatea de utilizatori.

Pentru plata cursurilor, platforma integrează sistemul de plăți Stripe, oferind utilizatorilor opțiuni diverse de plată. De asemenea, este implementată o funcționalitate de gestionare a comisioanelor și a plăților către instructori automat.

În plus față de funcționalitățile de bază, platforma permite implementarea altor funcționalități personalizate, adaptate nevoilor specifice ale proiectului.

## Dezvoltare Functionalitati:

### Autentificare si Acces Conditionat

- Implementare sistem de autentificare pentru utilizatori cu acces conditionat la anumite resurse.

### Gestionarea cursurilor prin panoul de administrare

- Creare panou de administrare pentru gestionarea cursurilor, facilitând actualizarea și gestionarea conținutului.

### Integrare Stripe cu 3 metode de plata diferite

- Integrare a sistemului de plăți Stripe pentru a oferi utilizatorilor opțiuni diverse de plată.

### Integrare AWS SES și serviciu de email marketing

- Implementare serviciu de email marketing prin AWS SES pentru comunicare eficientă cu utilizatorii.

### Securizarea videourilor împotriva descărcărilor neautorizate

- Implementare măsuri de securitate pentru a preveni descărcarea neautorizată a conținutului video.

### Hosting prin VPS

- Găzduirea site-ului și a aplicației pe un server virtual privat (VPS) pentru performanță și securitate crescută.

### Configurare Nginx

- Configurare server web Nginx pentru gestionarea traficului și securitatea aplicației.

### Configurare VPS (Ubuntu 20)

- Configurare și administrare a serverului virtual privat pe platforma Ubuntu 20.

### Controlul Versiunii cu Git / Github

- Utilizarea sistemului de control al versiunilor Git și platformei GitHub pentru gestionarea și colaborarea eficientă la cod.
